## 🇻🇳 Tiếng Việt

Ứng dụng gợi ý món ăn thông minh giúp bạn quyết định "Hôm nay ăn gì?" một cách dễ dàng và thú vị!

### 🌐 Live Demo
**[🍜 Thử ngay!](https://whattoeat-nine.vercel.app/)**

### ✨ Tính năng

- 🎯 Gợi ý món ăn ngẫu nhiên theo loại bạn chọn
- 🌤️ Gợi ý món ăn theo thời tiết hiện tại
- 🍽️ Tìm quán ăn gần đây với Google Maps
- ❤️ Lưu và quản lý món ăn yêu thích
- 🎨 Giao diện đẹp mắt với animation mượt mà
- 📱 Responsive design, tương thích mọi thiết bị
- 🔮 Hiệu ứng loading dễ thương
- 📤 Chia sẻ kết quả với bạn bè
- 🎲 Gợi ý món khác nhanh chóng

### 🚀 Cách chạy

1. Cài đặt dependencies:
```bash
npm install
```

2. Chạy development server:
```bash
npm run dev
```

3. Mở trình duyệt và truy cập: `http://localhost:5173`

### 🛠️ Công nghệ sử dụng

- **React 18** - UI Framework
- **TypeScript** - Type safety
- **Vite** - Build tool nhanh
- **Tailwind CSS** - Styling
- **Framer Motion** - Animation
- **Lucide React** - Icons

### 📁 Cấu trúc dự án

```
src/
├── components/     # React components
├── data/          # Dữ liệu món ăn
├── hooks/         # Custom hooks
├── types/         # TypeScript types
├── App.tsx        # Component chính
├── main.tsx       # Entry point
└── index.css      # Global styles
```

Sử dụng tính năng chia sẻ để gửi kết quả cho bạn bè qua:

## 🇺🇸 English

Smart food suggestion app that helps you decide "What to eat today?" easily and fun!

### 🌐 Live Demo
**[🍜 Try it now!](https://whattoeat-nine.vercel.app/)**

### ✨ Features

- 🎯 Random food suggestions based on your chosen category
- 🌤️ Weather-based food recommendations
- 🍽️ Find nearby restaurants with Google Maps
- ❤️ Save and manage favorite foods
- 🎨 Beautiful interface with smooth animations
- 📱 Responsive design, compatible with all devices
- 🔮 Cute loading effects
- 📤 Share results with friends
- 🎲 Quick suggestions for other dishes

### 🚀 How to run

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open browser and visit: `http://localhost:5173`

### 🛠️ Technologies used

- **React 18** - UI Framework
- **TypeScript** - Type safety
- **Vite** - Fast build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animation
- **Lucide React** - Icons

### 📁 Project structure

```
src/
├── components/     # React components
├── data/          # Food data
├── hooks/         # Custom hooks
├── types/         # TypeScript types
├── App.tsx        # Main component
├── main.tsx       # Entry point
└── index.css      # Global styles
```

### 🎨 Food categories
